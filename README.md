# Herb-Detection-System

**Herb-Detection-System** is a Python pipeline for detecting and identifying herbs in images using the EfficientDet model. It outputs bounding boxes and confidence scores for each herb, efficiently handling multiple species for applications like agriculture monitoring and botanical research.

---

## Features

- Detects and identifies herbs in images using **EfficientDet**  
- Provides bounding boxes and confidence scores for each detected plant  
- Handles multiple species and varying image conditions efficiently  
- Includes sample dataset and scripts for testing and inference  
- Modular design for easy customization and integration
